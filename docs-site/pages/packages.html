Package Manager

Manage dependencies with ease using the built-in package manager.

Quick Start

# Initialize a new project
raven pkg init

# Add a package
raven pkg add raven-ui

# Install all dependencies
raven pkg install

Core Commands

CommandDescription
raven pkg initInitialize package manifest
raven pkg add <name>Add a dependency
raven pkg installInstall all dependencies
raven pkg updateUpdate to latest versions
raven pkg treeShow dependency tree
raven pkg search <query>Search for packages
raven pkg publishPublish to registry

raven-ui

Production-ready UI component library with 10+ components.

raven pkg add raven-ui
import { Button, Input, Card } from "raven-ui"

<Card title="Welcome">
    <Input label="Name" />
    <Button variant="primary">Submit</Button>
</Card>

Components: Button, Input, Card, Modal, Dropdown, Tabs, Accordion, Tooltip, Badge, Spinner

raven-router

Client-side routing with nested routes and guards.

raven pkg add raven-router
use raven_router::{Router, Route, Link};

<Router>
    <Route path="/" component={Home} />
    <Route path="/about" component={About} />
</Router>

raven-http

HTTP client for making API requests.

raven pkg add raven-http
use raven_http::Http;

let data = Http::get("/api/users").await?;
Http::post("/api/users", user_data).await?;

raven-store

Advanced state management for complex applications.

raven pkg add raven-store
use raven_store::Store;

let store = Store::new(AppState { ... });
store.dispatch(Action::UpdateUser(user));

raven-forms

Form handling with validation and error management.

raven pkg add raven-forms
