ğŸ’» Using Packages in Your Code

Import Syntax:

Use standard ES6-style imports to use packages in your RavensOne code. 

// Import from installed package
import { HttpClient } from '@ravens/http-client';

// Use the imported module
const client = new HttpClient('https://api.example.com');

async fn main() {
    const response = await client.get('/users');
    console.log(response);
}

main();
// Import UI components
import { Button, Card, Modal } from '@ravens/ui-components';

// Create a button
const btn = new Button({
    label: 'Click Me',
    onClick: () => console.log('Clicked!')
});

btn.mount(document.querySelector('#app'));
